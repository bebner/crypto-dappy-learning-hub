## The mission brief

The backend is an integral part of a marketplace on Flow. Let's learn how to deploy our own backend on AWS! The starting branch is market-bonus-missions-deployment2

In order to deploy your own AWS serverless backend, you will need an AWS account and the AWS CLI. Please visit the links below if you need help setting these up:

**AWS Account Setup:**
https://aws.amazon.com/premiumsupport/knowledge-center/create-and-activate-aws-account/

**AWS CLI Setup:**
https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-getting-started.html
