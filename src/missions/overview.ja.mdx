転職おめでとうございます🎉 誰もが話題にしている次の大きなスタートアップにジョインしました。**CryptoDappy** です。
CEO の Rhea Dappy が率いるチームは、野心的な計画を持っています：ブロックチェーン上で、これまでに作られた最高のコレクターズプラットフォームを構築することです。
彼らは、自分たちのアプリケーションを多くの人に向けて、シームレスなユーザーエクスペリエンスで拡張したいと考えているため、構築するブロックチェーンとして Flow を選びました。
Flow では、ユーザーと開発者のための簡単なオンボーディング体験と組み合わさったスケーラビリティを得ることができます。もし、熟練した開発者を確保するのはそれほど難しくなければ！

そこで、あなたの出番です。あなたはこれまでの人生で 1、2 行のコードを書いたことはあっても、ブロックチェーンや Flow についてはほとんど知りません。
*しかし、それは変わります！* チームはあなたのために専用の学習課題を用意していますので、彼らのコレクティブル・ゲーム「CryptoDappy」 V1.0 に貢献しながら、
技術的な側面に素早く追いつくことができます。

## あなたが学ぶこと

このコースでは、Flow 上でブロックチェーンアプリケーションの構築を素早く始めるために必要なすべての知識を学びます。具体的には、以下の方法を学びます。

- ブロックチェーン・ウォレットでユーザーを認証する
- スクリプトによるブロックチェーンデータの照会
- トランザクションを使ってスマートコントラクトと対話する
- 独自のスマートコントラクトを作成、テスト、デプロイする
- 決済システムをアプリケーションに統合して、自分のデジタル資産を販売する方法を学ぶ


## あなたが作るもの

CryptoDappyは **コレクティブル・ゲーム** で、ユーザーはかわいい Dappy たちをストアで個別に購入するか、
**Dappy Drops** に参加して Dappy のパックを購入することで集めることができます。
ユーザーはいつでもアプリケーションにサインインし、自分の **Dappy コレクション** を見たり、以前に購入したパックを開いたりできます。

技術的な観点から見ると、CryptoDappy は、React フロントエンドと、
Flow のプログラミング言語である Cadence で書かれたスマートコントラクトのコレクションで構成されています。

コース終了時には、完全に機能するアプリケーションをテストネットにデプロイして、
ブロックチェーン開発者のポートフォリオに加えることができます。
また、独自のアプリケーションを書くためのスターターテンプレートとして使うこともできます。

## どうやって学ぶか

様々な **ミッションベースのシナリオ** で CryptoDappy を作ります。各ミッションには以下が含まれます。

- **ミッション概要**： ここでは、ミッションの成果と学習目標が一目でわかります。
- **学習コンテンツ**： このパートでは、ミッションで構築する機能に必要な知識基礎を得ることができます。
- **プロジェクトウォークスルー**： このパートでは、CryptoDappy プロジェクトでの実際の作業をフォローできます。

We'll build CryptoDappies in various **mission-based scenarios** that combine the 
theoretical aspects you need to know with hands-on code walkthroughs. Each mission contains:

- **The mission brief**: Here you'll find all the mission's outcomes and learning goals at one glance.
- **Learning content:** In this part you will get the knowledge foundation needed for the features you'll build in the mission
- **Project walkthrough:** This is the actual work on the CryptoDappy project that you can follow along


## どこで助けを求めるか？

プロジェクトの構築に行き詰まりを感じたり、コースに関する質問がある場合は、
専用の [Flow コミュニティフォーラム](https://forum.onflow.org/c/community-projects/cryptodappy/35) で助けを得ることができます。

## コンテンツ

### 🚀 [オンボーディング](/missions/mission-0): はじめに

スターターアプリケーションを知り、一般的なコンセプトを紹介します。

### 🔐 [ミッション #1](/missions/mission-1): 認証

Flow Client Library (FCL) を使って数行でブロックチェーン認証を追加します。


### 📜 [ミッション #2](/missions/mission-2): スクリプト

Cadence のスクリプトを使って、ブロックチェーンに問い合わせます。


### ✉️ [ミッション #3](/missions/mission-3): トランザクションとリソース

Dappy NFT のコレクションリソースを作成するための最初のトランザクションを書きます。


### 💰 [ミッション #4](/missions/mission-4): FUSD と Fungible Tokens

FUSD 決済をインテグレーションし、Flow における Fungible Tokens の基本を把握します。


### 💎 [ミッション #5](/missions/mission-5): NFT の mint

あなたの初めての Dappy NFT を mint して、コレクションに加えましょう。


### 📦 [ミッション #6](/missions/mission-6): パック

複数の Dappy を一度に mint するための仕組みを Dappy パックに実装します。
